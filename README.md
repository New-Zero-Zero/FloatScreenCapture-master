# FloatScreenCapture
”点击“悬浮图标截图，“长按”悬浮图标清除，
<br/>
<br/>
 悬浮于Android桌面的截图，可上下移动。用于记录一些App有临时闪现的图像进行截图记录，根据需要图片可保存到本地，特别是Ingress这个游戏Hack画图形简单快乐。
 <br/>
 <br/>
 目前支持到Android10
 <br/>
 <br/>
 手机实际使用情况
 <br/>
 <br/>
 只有小米10在使用中，其他品牌手机不清楚情况，有问题可提交lssues
 <br/>
 <br/>
 桌面上最多显示前5张图片，再点击截图勾选保存到相册的情况下会自动保存至相册不会在界面上显示。
  <br/>
